Retrieve History Data

GET

/v1.1/device/<device_id>/sensor/<sensor_id>.json?start=<start_timestamp>&end=<end_timestamp>&interval=<interval>&page=<page>

字段 类型 备注 device_d int 设备ID sensor_id int 传感器ID start_timestamp string 开始时间，格式：2014-5-10T11：40 end_timestamp string 开始时间，格式：2014-5-15T11：40 interval int 间隔时间 page int 分页，默认是1

字段 类型 备注
device_id int 设备ID
sensor_id int 传感器ID
start_timestamp string 开始时间，格式：2014-5-10T11：40
end_timestamp string 开始时间，格式：2014-5-15T11：40
interval int 间隔时间
page int 分页，默认是1

interval（数据采样间隔）说明

Interval 说明
1 每秒
10 10秒
